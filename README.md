## 🗒 About

These are Currency Converter app using Currency convert (ex. EURO to INR).

## ⚙️ Features

- Main features are as follows:

    - Splash Screen
    - Use Currency Api
    - CarouselSlider
    - Select functionality
    - Easy to Convert
